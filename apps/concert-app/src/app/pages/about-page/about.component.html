Case Description

In dit project is het mogelijk om tickets te kopen voor een concert. Een gebruiker kan concerten bekijken en hier meer informatie over zien. Daarnaast kan de gebruiker zien wat voor artiesten er allemaal zijn geregisteerd op het platform en welke kunnen aanmelden voor de projecten. Als de gebruiker inlogd kan hij daadwerkelijk tickets kopen voor een concert na keuzen. Verder kan hij zijn tickets in zien en zijn profiel bewerken. Als admin heb je meer functionaliteit. Zo kan je als admin concerten aanmaken en/of bewerken en hier artiesten aan koppelen. Daarnaast kan je ook nieuwe artiesten toevoegen aan het systeem. Verder is het mogelijk om gebruikers te beheren.
